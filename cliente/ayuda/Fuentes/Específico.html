Registro Específico contiene
Código interno de proceso

Cada vez que se genera un proceso éste debe ser identificado para diferenciarlo del mismo tipo de proceso de otros expedientes
o ejecuciones del procedimiento.

El código del proceso tendrá la siguiente estructura:

AAAAMMDD-HHMMSS-rrrrrr

Donde:

  • AAAAMMDD - Fecha en que se genera
  • HHMMSS - Hora en que se genera
  • RRRRRR - Número correlativo de seis cifras que guarda la secuencia de generación de procesos.
    Este número será correlativo a todos los procesos que se generan, de manera que guarden
    la secuencia de generación de los procesos sean del tipo que sean.
Código de procedimiento

Código del primer proceso (proceso 0 del procedimiento, que siempre existe en todo procedimiento y que se genera
cuando se inicia una instancia del procedimiento). Este número de proceso definirá la trazabilidad hacia adelante y hacia atrás dentro de un procedimiento.
Cod. Contratación En este caso será el código único para cada trabajador cuando sea agregado a la empresa.
Documentación Vinculada

Lista de documentos (físicos o digitales) relacionados con el proceso.

  1. Código de documento o enlace al mismo si es digital
  2. Tipo y descripción del tipo de documento

Si la lista de documentos es fija, la descripción de los mismos [b)] iría a este nivel,
si depende de los datos que se generan en el evento en que se produce el proceso, irán en el atributo [3] del proceso especifico.

Nota:
Recomendamos todos los procesos digitales. Y si el proceso es papel, se genera un proceso digital que referencia al de documento.

Generador del Proceso

Persona o empresa externa o sistema que genera el proceso.

Esta información podría estar a nivel de proceso específico si puede variar el agente que genera el proceso [4].
Lo habitual es que sea siempre el generador de proceso sea siempre mismo y este a este nivel.
Autorizados para su consulta

Persona, empresa externa o sistema autorizados para la consulta del proceso.

Esta información podría estar a nivel de proceso específico si puede variar el/los agente/s que consulta/n el proceso [5].
Lo habitual es que sea siempre el mismo y este a este nivel.
