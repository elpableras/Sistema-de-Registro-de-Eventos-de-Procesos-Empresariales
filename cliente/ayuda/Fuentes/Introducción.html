Sistema de Registro

Está aplicación, llamada Sistema de Registro de Eventos de Procesos Empresariales permite dar soporte a la trazabilidad de procesos, tanto de los que dan lugar a productos tangibles como intangibles.

El programa comienza con la pantalla de bienvenida al usuario, dándole a elegir las opciones:

  • Administrador de Procesos
  • Aplicación
  • Crear Registros Automáticos
  • Salir


Administrador de Procesos

Para que aparezcan los elementos para la introducción del nombre de usuario y la contraseña, se debe teclear una secuencia de teclas.


Una vez, introducido el usuario y la contraseña se enviarán al Servidor, para confirmar que existe ese cuenta administrador asociada a esa contraseña en la BBDD, sino es así, se producirá un error, también para mandar los datos del usuario y contraseña, se necesita saber una secuencia de teclas.


Aplicación

Una vez, introducido el usuario y la contraseña en sus campos correspondientes, se enviarán al Servidor, para confirmar que existen esos usuarios asociados con esas contraseñas en la BBDD, una vez confirmados los datos se enviara al usuario a la pantalla correspondiente del usuario introducido.


Crear Registros Automáticos

La última opción disponible, es la de crear automáticamente los registros, tanto los que debe generar el Administrador de Procesos como los de la Aplicación.

Simula la creación de los procesos sigueintes:

  • Administrador de Procesos
    • Administrador
  • Aplicación
    • RRHH
    • Trabajador*
    • Gerente

Salir

El botón salir termina con la aplicación.


* El usuario y la contraseña serán asignados por la persona de RRHH.
