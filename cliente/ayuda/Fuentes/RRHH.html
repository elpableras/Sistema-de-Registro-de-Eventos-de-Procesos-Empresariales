RRHH

La pantalla de RRHH está dividida en varias secciones:

  1. Barra de Menú
  2. Barra de Herramientas
  3. Cuerpo Central
    1. Parte izquierda para las opciones sobre los procesos
    2. La parte derecha mostrara la información sobre el proceso, en este caso para el registro Específico de RRHH
  4. Barra de Estado

Barra de Menú

En la barra de menú podrá encontrar varios menús desplegables, entre ellos el de "Archivo", "Procedimiento" y "Ayuda".

Archivo

Cada uno de ellos con diferentes opciones, en el de Archivo tendrá las acciones de Imprimir y de Salir, los dos con teclas de acceso rápido, "ESC" para salir y "CTRL+P", para imprimir.

Procedimiento

En este tendrá una única acción la de Crear Procedimiento desplegara a su vez una lista de procedimientos que tenga cargados, cada uno de ellos tendrá diferentes pasos para crear los procesos Específicos de RRHH.

Ayuda

"Ayuda", que es el última opción, con acceso rápido pulsando sobre "F1", se encarga de traernos hasta aquí y de darnos información sobre la aplicación y el "Acerca de" nos da información acerca de la pantalla.


Barra de Herramientas

Está barra contiene dos pequeños botones, para agilizar algunas acciones: la creación de procesos anteriormente mencionada y un botón de impresión rápida, para la pantalla.


Cuerpo Central

El cuerpo central, está dividido en dos secciones, en el de la izquierda será para las acciones sobre los procesos, donde se podrá:

  1. Opciones: Ejecutar sobre el proceso las opciones que se despliegan
  2. Botón "Generar", es la creación del comienzo de dar de alta a un nuevo trabajador, este creará a la derecha el registro con los campos rellenados, se le pedirá a la persona de RRHH que agregue un nombre de usuario y una contraseña, para que el trabajador tenga acceso a la aplicación
  3. Botón "Ver Pendientes"Si hay procesos esperando ser tratados por el personal de RRHH, aparecerán en la lista de contratos después de pulsar sobre el botón, este seleccionara cada uno de ellos y pulsando sobre el botón "Seleccionar", creará el registro, donde podrá elegir agregar la documentación que se a necesaria cuando le aparezca la ventana correspondiente, después de aceptar, le aparecerá a la derecha el registro con los campos completados que deberá confirmar, para enviarlos a archivar.

Por último tiene el botón de "Aceptar", que se utiliza para cuando se creó el registro, aceptar la creación y enviarlo a archivar.


Barra de Estado

En esta sección es donde aparecerán los mensajes de las operaciones que se vayan realizando.
